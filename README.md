# TypeScript CLI Calculator

A simple Node.js TypeScript CLI app that implements some basic calculator features.

**Usage:**

Accepts math input to compute. `"[number1] [operator: + - * /] [number2]"`.

**Example:**

```shell
npm start "123 + 4"
```

**Run tests once:**

```shell
npm test
```

**Run tests in watch mode:**

```shell
npm run testing
```
